********
Glossary
********

.. glossary::

    Builder
        A :term:`Sphinx` builder compiles documentation into various
        output formats.

        .. seealso:: https://www.sphinx-doc.org/en/master/usage/builders/index.html

    CMake
        CMake is a compilation configuration platform that manages the build
        process in an operating system and in a compiler-independent manner.

        .. seealso:: https://cmake.org/overview/

    Doxygen
        Doxygen is a documentation generator for writing software reference
        documentation from annotated source code. It is often used to create
        documentation from a C/C++ codebase.

        .. seealso:: https://www.doxygen.nl/

    find_package
        :term:`CMake` function is used to provide locations of package
        dependencies to build a project.

        .. seealso:: https://cmake.org/cmake/help/latest/command/find_package.html

    Pip
        Pip is a package-management system written for Python.

        .. seealso:: https://pip.pypa.io/en/stable/

    reStructuredText
        Lightweight markup language.

        .. seealso:: https://docutils.sourceforge.io/rst.html

    Sphinx
        Python documentation generator which converts :term:`reStructuredText`
        files into HTML and other formats.

        .. seealso:: https://www.sphinx-doc.org/
